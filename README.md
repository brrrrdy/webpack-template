# Webpack-Template

Template for future projects using webpack created by me.

## Useful Commands

| Command                                      |     Function      |
| -------------------------------------------- | :---------------: |
| npm start                                    | npx webpack serve |
| npm run build                                | production build  |
| tree -I node_modules > project-structure.txt | project structure |
